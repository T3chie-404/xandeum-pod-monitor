Health Score Formula

Overall score = 40% System + 60% Services.

  • System score starts at 100, reduces based on resource usage
  • CPU: -20 if ≥91%, -10 if ≥71%
  • Memory: -20 if ≥91%, -10 if ≥76%
  • Disk: -30 if ≥91%, -15 if ≥81%
  • Service score: (running services / total) × 100
  • Overall = (System × 0.4) + (Services × 0.6)
  • Status: Healthy ≥80, Warning ≥50, Critical <50
